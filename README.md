This project was initially inspired from Prescript in LoR.
For now you're able to :
- Receive 1 directive everyday (Based on UTC timezone)
- mark it as executed
- depending on your progression, the UI turns more red (7/7 the UI is crimson)
- The directives now have an intensity/difficulty level (day 1 is far easier than day 7)
- You can now clear the cache to debug if anything happens, note that it will reset your weekly progress

There will be more directives coming later on, the updates will be weekly since otherwise they'd make everyone lose progress.
